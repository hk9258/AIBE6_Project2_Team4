# 📋 Java CLI 텍스트 게시판

Java 콘솔 기반의 텍스트 게시판 프로그램입니다.
게시글 작성·조회·수정·삭제와 검색, 파일 저장/불러오기 기능을 포함합니다.

---

## 📁 파일 구조

```
bulletin-board/
├── src/
│   ├── Main.java      ← 진입점 (App 실행)
│   ├── App.java       ← 핵심 로직 (CRUD + 검색 + 파일 I/O)
│   ├── Article.java   ← 게시글 데이터 클래스
│   └── Rq.java        ← 명령어 파싱 유틸 클래스
└── README.md
```

> 프로그램 실행 후 `exit` 명령어를 입력하면 같은 디렉토리에 `articles.dat` 파일이 자동 생성됩니다.

---

## ▶ 실행 방법

```bash
# 1. 컴파일
javac src/*.java -d out

# 2. 실행
java -cp out Main
```

## 🧩 명령어 목록

| 명령어            | 설명                            |
|-------------------|---------------------------------|
| `write`           | 게시글 작성                     |
| `list`            | 게시글 목록 보기 (최신순)       |
| `detail [번호]`   | 게시글 상세 보기 + 조회수 증가  |
| `update [번호]`   | 게시글 제목/내용 수정           |
| `delete [번호]`   | 게시글 삭제 (확인 절차 포함)    |
| `search [키워드]` | 제목·내용 포함 키워드 검색      |
| `help`            | 명령어 도움말 출력              |
| `exit`            | 데이터 저장 후 프로그램 종료    |

## 🔁 전체 실행 흐름

```
main()
  └── App.run()
        ├── loadFromFile()   // 저장된 데이터 불러오기
        └── while(true)      // 입력 대기 루프
              ├── Rq로 입력 파싱
              ├── switch로 명령어 분기
              │     ├── write  → writeArticle()
              │     ├── list   → listArticles()
              │     ├── detail → showDetail()
              │     ├── update → updateArticle()
              │     ├── delete → deleteArticle()
              │     └── search → searchArticles()
              └── exit → saveToFile() → 종료
```

## ✅ 구현 기능 목록

- [x] 게시글 작성 (`write`)
- [x] 게시글 목록 최신순 출력 (`list`)
- [x] 게시글 상세보기 (`detail [번호]`)
- [x] 게시글 수정 (`update [번호]`)
- [x] 게시글 삭제 — 확인 절차 포함 (`delete [번호]`)
- [x] 조회수 기능 — `detail` 시 자동 증가, `list`에서 표시
- [x] 게시글 검색 — 제목·내용 포함 검색 (`search [키워드]`)
- [x] 파일 저장/불러오기 — `exit` 시 저장, 시작 시 자동 로드
- [x] 입력 유효성 검사 — 빈 제목·내용 방지, 잘못된 ID 처리

## 🧪 테스트 (TDD)

본 프로젝트는 **TDD(Test Driven Development)** 방식을 적용하여 개발했습니다.  
기능 구현 전에 테스트 코드를 먼저 작성하고, 실패를 확인한 뒤 최소 구현으로 테스트를 통과시키고, 이후 리팩터링을 진행했습니다.

### 🔁 TDD 워크플로

- 테스트 먼저 작성 (실패)
- 최소 구현으로 테스트 통과
- 리팩터링 (가독성 및 구조 개선)
